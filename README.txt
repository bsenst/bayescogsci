Data and code for An Introduction to Bayesian Data Analysis for Cognitive Science by Nicenboim, Schad, Vasishth

The data are available from the bcogsci package in this github repository: 

https://github.com/bnicenboim/bcogsci

Install the bcogsci library by following the instructions in the above github repo.

The source code for the book will become available once the book is in its final form (expected: end-2022).
